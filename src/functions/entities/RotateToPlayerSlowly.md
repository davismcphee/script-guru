# RotateToPlayerSlowly(e, v)

| Parameter | Type   | Description                  |
| --------- | ------ | ---------------------------- |
| e         | number | The entity ID                |
| v         | number | The speed at which to rotate |

Rotate the target entity toward the player at "v" speed.
