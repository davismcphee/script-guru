# ChangePlayerWeapon(str)

| Parameter | Type   | Description     |
| --------- | ------ | --------------- |
| str       | string | The weapon name |

## Details

Change the player's weapon based on the provided weapon name.

## Example

```lua
ChangePlayerWeapon("myWeapon")
```
