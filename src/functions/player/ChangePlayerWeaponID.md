# ChangePlayerWeaponID(id)

| Parameter | Type   | Description   |
| --------- | ------ | ------------- |
| id        | number | The weapon ID |

## Details

Change the player's weapon based on the provided weapon ID.

## Example

```lua
ChangePlayerWeaponID(1)
```
