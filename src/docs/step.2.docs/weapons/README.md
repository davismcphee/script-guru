# Weapons Globals

GameGuru exposes these globals to access weapon properties:

```lua
g_projectileevent_damage

g_projectileevent_entityhit

g_projectileevent_explosion

g_projectileevent_name

g_projectileevent_radius

g_projectileevent_x

g_projectileevent_y

g_projectileevent_z

g_WeaponAmmo

g_WeaponClipAmmo

g_WeaponPoolAmmo

g_WeaponSlotGot

g_WeaponSlotPref

weapon_name
```
