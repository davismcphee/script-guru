```lua
mp_enemiesLeftToKill

mp_friendlyfireoff

mp_gameMode

mp_isConnectedToSteam

mp_isServer

mp_me

mp_playerConnected

mp_playerDeaths

mp_playerKills

mp_playerNames

mp_playerTeam

mp_servertimer

mp_showscores

mp_teambased
```
